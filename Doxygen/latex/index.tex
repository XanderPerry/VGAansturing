This project has been finished on 13/01/2026\hypertarget{index_intro_sec}{}\doxysection{\texorpdfstring{Introduction}{Introduction}}\label{index_intro_sec}
Welcome to the VGA Display Terminal documentation. This project utilizes an {\bfseries{STM32\+F407\+GTx}} to drive a VGA screen. The system operates as a serial terminal\+: it receives ASCII string commands via UART2 and renders graphics accordingly.\hypertarget{index_arch_sec}{}\doxysection{\texorpdfstring{Architecture}{Architecture}}\label{index_arch_sec}
The software is divided into three distinct layers\+:


\begin{DoxyEnumerate}
\item {\bfseries{Front Layer}}\+: Parses incoming comma-\/separated strings (e.\+g., {\ttfamily lijn,0,0,100,100,rood}).
\item {\bfseries{Logic Layer}}\+: Mathematical implementations of drawing algorithms (Bresenham lines, Midpoint circles).
\item {\bfseries{I/O Layer}}\+: Hardware interface for UART interrupts and VGA pixel pushing.
\end{DoxyEnumerate}

\hypertarget{index_usage_sec}{}\doxysection{\texorpdfstring{Hardware \& Software Setup}{Hardware \& Software Setup}}\label{index_usage_sec}
To use the VGA Display Terminal, you need to set up the motherboard connections and configure a serial terminal on your PC.\hypertarget{index_hw_req}{}\doxysubsection{\texorpdfstring{Hardware Requirements}{Hardware Requirements}}\label{index_hw_req}

\begin{DoxyItemize}
\item {\bfseries{STM32\+F407G Board}} mounted on the VGA Motherboard.
\item {\bfseries{VGA Monitor}} and VGA Cable.
\item {\bfseries{USB Cables}} (One for flashing/debug, one for UART communication).
\end{DoxyItemize}\hypertarget{index_connect_sec}{}\doxysubsection{\texorpdfstring{Connections}{Connections}}\label{index_connect_sec}
The motherboard handles the physical signal routing. Follow these steps\+:


\begin{DoxyEnumerate}
\item {\bfseries{VGA}}\+: Connect the VGA Monitor to the VGA port on the motherboard.
\item {\bfseries{UART}}\+: Connect the motherboard\textquotesingle{}s {\bfseries{UART USB port}} to your PC.
\item {\bfseries{Flashing}}\+: Connect the board\textquotesingle{}s {\bfseries{ST-\/\+Link/\+Programming USB port}} to your PC.
\end{DoxyEnumerate}\hypertarget{index_term_setup}{}\doxysubsection{\texorpdfstring{Terminal Configuration}{Terminal Configuration}}\label{index_term_setup}

\begin{DoxyEnumerate}
\item Open a terminal program (e.\+g., {\bfseries{Tera Term}}, {\bfseries{Pu\+TTY}}, or {\bfseries{Real\+Term}}).
\item Select the COM port associated with the {\bfseries{UART connection}} (not the ST-\/\+Link).
\item Configure the serial port with the following settings\+:
\begin{DoxyItemize}
\item {\bfseries{Baud Rate\+:}} 115200
\item {\bfseries{Data Bits\+:}} 8
\item {\bfseries{Parity\+:}} None
\item {\bfseries{Stop Bits\+:}} 1
\item {\bfseries{New Line\+:}} CR+\+LF (Ensure sending {\ttfamily \textbackslash{}n} on Enter)
\end{DoxyItemize}
\end{DoxyEnumerate}\hypertarget{index_send_cmd}{}\doxysubsection{\texorpdfstring{Sending Commands}{Sending Commands}}\label{index_send_cmd}
Once connected, the screen may appear black or white depending on initialization.
\begin{DoxyEnumerate}
\item Click into the terminal window.
\item Type a command string (e.\+g., {\ttfamily clearscherm,wit}).
\item Press {\bfseries{Enter}}.
\item The parser expects a newline character ({\ttfamily \textbackslash{}n}) to execute the command.
