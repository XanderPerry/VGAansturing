Draws a filled circle on the VGA display. 

This function uses the midpoint circle algorithm to draw a filled circle centered at the specified coordinates. The circle is filled by drawing horizontal spans between symmetric points on each iteration.

This function implements Bresenham\textquotesingle{}s line algorithm to efficiently draw a line between two points using only integer arithmetic. Line thickness is achieved by drawing a filled circle (brush) at each point along the path.

This function calls \doxylink{_a_p_i___l_i_b_8c_a970082c425e5df1b713a39b66adbf37a}{API\+\_\+draw\+\_\+circle()} to simulate line thickness. For very large weights, performance may decrease due to repeated circle drawing.
